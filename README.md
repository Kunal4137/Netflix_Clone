# 🎬 Netflix Clone

A pixel-accurate front-end clone of Netflix's homepage built with pure HTML, CSS, and JavaScript — 
no frameworks, no shortcuts.

## 🔗 Live Demo
👉 [View Live](https://6a3248a3478f2ac6983778b8--jazzy-kitten-38b07d.netlify.app/)

## ✨ Features
- Responsive hero banner section
- Multi-row content carousels
- Hover animations on cards
- Mobile-friendly layout using CSS Flexbox & Grid
- Netflix-style footer with links

## 🛠️ Tech Stack
HTML5 · CSS3 · JavaScript (Vanilla)
